# Global Wallpaper Overlay - 全局壁纸遮罩

一款适用于 Chrome 内核浏览器的扩展插件，允许用户上传自定义图片作为网页的全局壁纸遮罩。

## 功能特性

- **图片上传**: 支持 JPG、PNG、GIF、WebP 等常见图片格式
- **图片裁剪**: 内置裁剪工具，支持自由裁剪和预设比例
- **透明度控制**: 滑动条调节壁纸透明度 (0-100%)
- **两种显示模式**:
  - **悬浮遮罩**: 图片悬浮在网页内容上方
  - **网页背景**: 图片作为网页的背景
- **全局应用**: 在所有网页上生效
- **实时预览**: 即时查看设置效果

## 安装方法

### 开发者模式安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

### 从 Chrome 网上应用店安装

*即将推出*

## 使用说明

1. **上传壁纸**
   - 点击浏览器工具栏中的扩展图标
   - 在弹出窗口中点击上传区域
   - 选择您喜欢的图片

2. **裁剪图片**
   - 上传后，点击"裁剪图片"按钮
   - 调整裁剪区域
   - 点击"确认裁剪"保存

3. **调整设置**
   - 使用滑动条调节透明度
   - 选择显示模式（悬浮遮罩/网页背景）
   - 点击"应用设置"保存

4. **启用/禁用**
   - 使用开关按钮启用或禁用遮罩

## 技术架构

### 文件结构

```
Global Wallpaper Overlay/
├── manifest.json              # 扩展清单文件 (Manifest V3)
├── src/
│   ├── popup/
│   │   ├── popup.html         # 弹出窗口界面
│   │   ├── popup.css          # 弹出窗口样式
│   │   └── popup.js           # 弹出窗口逻辑
│   ├── content/
│   │   ├── content.js         # 内容脚本 (注入到网页)
│   │   └── content.css        # 内容脚本样式
│   ├── background/
│   │   └── background.js      # 后台服务脚本
│   ├── crop/
│   │   ├── crop.html          # 裁剪页面
│   │   ├── crop.css           # 裁剪页面样式
│   │   └── crop.js            # 裁剪功能逻辑
│   ├── icons/
│   │   ├── icon16.png         # 16x16 图标
│   │   ├── icon32.png         # 32x32 图标
│   │   ├── icon48.png         # 48x48 图标
│   │   └── icon128.png        # 128x128 图标
│   └── styles/
│       └── content.css        # 全局内容样式
├── package.json               # 项目依赖
└── README.md                  # 项目说明
```

### 核心技术

- **Manifest V3**: 最新的 Chrome 扩展清单格式
- **Chrome Storage API**: 本地存储用户设置和图片数据
- **Content Scripts**: 注入到所有网页，实现壁纸显示
- **Cropper.js**: 图片裁剪功能
- **Service Worker**: 后台处理和状态同步

### 通信流程

```
用户操作 → Popup → Storage → Background → Content Script → 网页显示
```

1. 用户在弹出窗口中调整设置
2. 设置保存到 Chrome Storage
3. Background 脚本监听存储变化
4. 向所有标签页的 Content Script 发送消息
5. Content Script 更新壁纸显示

## 常见问题

### Q: 为什么某些网站看不到壁纸？

A: 某些网站可能使用了极高的 z-index 值或特殊的安全策略。我们的扩展使用了最大 z-index 值 (2147483647)，但仍有少数网站可能无法显示。

### Q: 图片上传后为什么没有显示？

A: 请检查：
- 扩展是否已启用
- 是否点击了"应用设置"
- 刷新当前页面

### Q: 如何完全移除壁纸？

A: 在弹出窗口中点击壁纸预览图上的删除按钮，或关闭扩展开关。

### Q: 支持哪些图片格式？

A: 支持所有浏览器支持的图片格式，包括 JPG、PNG、GIF、WebP、BMP 等。

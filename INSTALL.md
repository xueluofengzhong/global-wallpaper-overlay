# 安装指南 - Global Wallpaper Overlay

## 快速安装 (开发者模式)

### 步骤 1: 打开扩展管理页面

在 Chrome 浏览器中访问:
```
chrome://extensions/
```

### 步骤 2: 启用开发者模式

点击右上角的 **"开发者模式"** 开关，确保它处于开启状态。

### 步骤 3: 加载扩展

1. 点击 **"加载已解压的扩展程序"** 按钮
2. 在文件选择对话框中，选择 `Global Wallpaper Overlay` 文件夹
3. 点击 **"选择文件夹"**

### 步骤 4: 验证安装

安装成功后，您应该能在浏览器工具栏看到扩展图标 (一个渐变色的房屋图标)。

## 使用方法

### 首次使用

1. **点击扩展图标** - 在浏览器工具栏中找到并点击扩展图标
2. **上传图片** - 在弹出窗口中点击上传区域，选择您喜欢的图片
3. **裁剪图片** (可选) - 点击"裁剪图片"按钮进行裁剪
4. **调整设置**:
   - 使用滑动条调节透明度 (0-100%)
   - 选择显示模式:
     - **悬浮遮罩**: 图片悬浮在网页内容上方
     - **网页背景**: 图片作为网页的背景
5. **应用设置** - 点击"应用设置"按钮保存

### 功能说明

#### 透明度控制
- 0%: 完全透明 (不可见)
- 100%: 完全不透明 (完全遮挡网页内容)
- 推荐设置: 30-50% (既能显示壁纸，又不影响阅读)

#### 显示模式

**悬浮遮罩模式**
- 图片悬浮在所有网页内容上方
- 适合: 显示艺术图片、水印
- 特点: 不影响网页布局

**网页背景模式**
- 图片作为网页的背景
- 适合: 自定义网页背景
- 特点: 网页内容在图片上方

#### 裁剪功能
- 支持自由裁剪
- 支持预设比例 (16:9, 4:3, 1:1)
- 支持旋转和缩放
- 实时预览

## 常见问题

### Q: 扩展图标没有出现？
A: 
1. 检查是否启用了开发者模式
2. 确认扩展已正确加载 (扩展管理页面显示"已启用")
3. 尝试刷新扩展管理页面

### Q: 上传图片后没有显示？
A:
1. 确认扩展已启用 (开关按钮为开启状态)
2. 检查是否点击了"应用设置"
3. 刷新当前页面
4. 检查图片格式是否支持 (JPG, PNG, GIF, WebP)

### Q: 某些网站看不到壁纸？
A:
- 某些网站使用了特殊的安全策略
- Chrome 内部页面 (chrome://) 不支持扩展
- 尝试在其他网站测试

### Q: 如何完全移除壁纸？
A:
1. 点击扩展图标
2. 点击壁纸预览图上的删除按钮 (×)
3. 或者关闭扩展的启用开关

### Q: 裁剪功能打不开？
A:
1. 确保已先上传图片
2. 检查浏览器是否阻止了弹出窗口
3. 允许扩展显示弹出窗口

## 卸载扩展

1. 访问 `chrome://extensions/`
2. 找到 "Global Wallpaper Overlay"
3. 点击 **"移除"** 按钮
4. 确认卸载

## 更新扩展

1. 下载最新版本的扩展文件
2. 访问 `chrome://extensions/`
3. 找到 "Global Wallpaper Overlay"
4. 点击刷新按钮 (圆形箭头图标)
5. 或者删除旧版本，重新加载新版本
